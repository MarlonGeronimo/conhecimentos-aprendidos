HTML - Hyper Text Markup Language

CSS - Cascading Style Sheets

A linguagem HTML é uma linguagem focada em conteúdo.

HTML

Conteúdo: textos, imagens, vídeos, tabelas.

CSS

Design: Cores, sombras, tamanhos, posicionamento.

JS

Interações: Menus, animações, popups, validações.

Conteúdo em HTML

<h1>Exemplo de título</h1>

<p>Exemplo de paragrafo</p>

<img src= "foto.png" Alt= "Exemplo de foto">

Estilo em CSS

h1 (seletor) {
font-family: Arial;
font-size: 20pt;
color: blue;
}

Estrutura básica de documento HTML

<!DOCTYPE html>
<html lang= "pt-br">
<head>
<meta charset= "UTF-8">
<meta name= "viewport" content= "width= device-width, initial-scale= 1.0">
<title>Document</title>
</head>
<body>
<h1>Olá, Mundo!</h1>
</body>
</html>

(Apertar "!" Ele deixa o código "pronto" para editar)

Paragrafos e quebra linhas

& lt; + & gt; Usar quando quiser mostrar os <> sem ser comando.

Adicionar emoji

&#x "código do emojipidia"

Texte com imagens

img src= "origem" alt= "explicar o que é a imagem"

(ctrl + space entre as aspas aparece as opções de imagem)

Mudar icon da página

Icon Archive= escolher um pronto.
favicon.cc= fazer um.
favicon.io= escolha uma forma e a IA faz por você.
Para colocar no código: dentro do head, em cima do title, digite "Link: favicon".

Hierarquia de títulos

H1= título principal - título de nível 1
H2= título de nível 2
h3= título de nível 3
Vai do H1 até o H6

Negrito e Itálico do jeito certo

Negrito maneira NÃO semântica: <b></b>
Negrito maneira semântica: <strong></strong>
Itálico maneira NÃO semântica: <i></i>
Itálico maneira semântica: <em></em>
