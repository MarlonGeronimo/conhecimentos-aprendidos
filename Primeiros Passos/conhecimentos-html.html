Coisas que aprendi em HTML

HTML - Hyper Text Markup Language.

Conteúdo: textos, imagens, vídeos, tabelas.

CSS - Cascading Style Sheets.

Design: Cores, sombras, tamanhos, posicionamento.

JS - JavaScript

Interações: Menus, animações, popups, validações.

Conteúdo em HTML

<h1>Exemplo de título</h1>

<p>Exemplo de paragrafo</p>

<img src= "foto.png" Alt= "Exemplo de foto">

Estrutura básica de documento HTML

<!DOCTYPE html>
<html lang= "pt-br">
<head>
<meta charset= "UTF-8">
<meta name= "viewport" content= "width= device-width, initial-scale= 1.0">
<title>Document</title>
</head>
<body>
<h1>Olá, Mundo!</h1>
</body>
</html>

(Apertar "!" Ele deixa o código "pronto" para editar)

Paragrafos e quebra linhas

<br> Ele quebra a linha pra baixo.
& lt; + & gt; Usar quando quiser mostrar os <> sem ser comando.

Adicionar emoji

&#x "código do emojipidia"

Texte com imagens

img src= "origem" alt= "explicar o que é a imagem"

(ctrl + space entre as aspas aparece as opções de imagem)

Mudar icon da página

Icon Archive= escolher um pronto.
favicon.cc= fazer um.
favicon.io= escolha uma forma e a IA faz por você.
Para colocar no código: dentro do head, em cima do title, digite "Link: favicon".

Hierarquia de títulos

H1= título principal - título de nível 1
H2= título de nível 2
h3= título de nível 3
Vai do H1 até o H6
(<p>Lorem</p> GERADOR DE BLABLABLA)

Negrito e Itálico do jeito certo

Negrito maneira NÃO semântica: <b></b>
Negrito maneira semântica: <strong></strong>
Itálico maneira NÃO semântica: <i></i>
Itálico maneira semântica: <em></em>
(ctrl + shift + p = dicas)

Formatações adicionais

<mark></mark> Adiciona como se fosse marca texto.
<mark style= "background-color:'cor'";> Muda a cor de maneira pontual.
Para mudar a cor de todos os mark, colocar na área "head" com <mark style= "background-color:'cor'";>

Texto grande: <big></big>
Texto Pequeno: <small></small>

Texto excluído: <del></del> O texto aparece riscado.
Texto inserido: <ins></ins> O texto aparece sublinhado.

Texto sobrescrito: <sup></sup> A parte selecionada no texto fica em cima.
Texto subscrito: <sub></sub> A parte selecionada no texto fica em baixo.

Códigos e citações

Código-fonte/Pré-formatação:

<code></code> Fica numa formatação melhor para código.
<pre></pre> Fica numa formatação com linha de código.
(Dá para usar as duas maneiras juntas.)

Citação simples:

<q></q> Coloca aspas (fica um sentido ao inves de colocar as aspas manualmente).

Citação Completa:

<blockquote></blockquote> A citação vai mais para a direita.

Abreviações:

<abbr title= "Exemplo de abreviação"></abbr> Serve para mostrar o que significa a abreviação.

Listas

<ol></ol> Listas ordenadas (1, 2, 3...)
(ol type aparece as opções de como você quer numerar)
<li></li> Item de lista (Ele que coloca os números)
<ul></ul> Listas não ordenadas (Não precisa seguir uma ordem)
(ul type você escolhe entre círculo preto: "disc"; círculo limpo: "circle"; um quadradinho: "square")
Para fazer uma lista dentro da lista é só colocar dentro do "<li>" a tag &"lt;ol>" ou "<ul>".

Listas de definição

<dl></dl> Para fazer a lista.
<dt></dt> Como se fosse o termo.
<dd></dd> Como se fosse a descrição.

Links

Link externo: Um link que não é nosso.
<a href= "Link do site"></a>
target= "_blank" rel= "external" O link abre em outra página em outra aba.
(Recomendado ir para outra página quando o link for externo e permanecer na aba quando for seu.)

Link interno:
Basicamente a mesma coisa do externo, mas você usa o seu link e use a dica do "ctrl + space" para abrir os links e pastas.

Link para download:
Ex:
<a href= "livro/meulivro.pdf" download= "meulivro.pdf" type= "application/pdf"></a>

Imagens dinâmicas

Dimensões:
No celular: 300/300 e 80/80
No tablet: 700/700 e 80/80
Notebook/TV: 1000/1000 e 80/80

<picture></picture> Essa tag permite que você crie várias sourcers, fontes e etc.

Ex:
<picture>
<Source media= "(max= width: 750px)" srcset= "imagens/foto-p.png" type= "image/png">
<Source media= "(max= width: 1050px)" srcset= "imagens/foto-m.png" type= "image/png">
<img src= "imagens/foto-g.png" alt= "Imagem flexível">
</picture>

Áudios

Ex:
<audio src= "media/happy-mistake.mp3" controls autoplay></audio>
Essa é a maneira simples de fazer.

Maneira "mais complexa"
Ex:
<audio preload= "metadata" autoplay controls loop>
<source src= "midia/guanacast-33.mp3" type= "audio/mpeg">
<source src= "midia/guanacast-33.ogg" type= "audio/ogg">
<source src= "midia/guanacast-33.wav" type= "audio/wav">
Caso não carregue nenhum, coloque o link do arquivo para a pessoa baixar.

Vídeos

Ex:
<video src= "midia/meu-video.mp4" width= "500" controls></video>
Essa é a maneira simples de fazer.

A maneira "mais complexa" é basicamente a mesma coisa, só muda o tipo (mp4, ogv, webm) colocar do mais leve para o mais pesado.
(Da pra colocar a tumb do vídeo com a tag "poster" dentro de vídeo)
Isso é hospedando ele do próprio site.

Vídeos do YouTube

No vídeo, clique em compartilhar e depois em incorporar, irá abrir o código e é só copiar e colar no seu próprio código.
